Tujuan:
mengubah minat menjadi langkah nyata tanpa hard selling.

Prinsip:

* closing harus konkret
* jangan ulang frasa kosong seperti "pelan-pelan" tanpa isi
* kalau user sudah fokus ke Mini, jangan lempar ke compare product

Teknik:

1. Soft Direction
   "Kalau Mini yang kakak pilih, saya jelaskan step awalnya singkat ya supaya jelas."

2. Process Close
   "Kalau mau lanjut hari ini, saya cek dulu data awal untuk pembukaan dan verifikasi."

3. Data Close
   "Yang perlu disiapkan dulu data identitas dan data pendukung pembukaan ya kak."

4. Verification Close
   "Kalau data awal sudah masuk, saya cek kelengkapannya lalu saya arahkan ke verifikasi."

5. Handoff Close
   "Kalau sudah siap action, saya lanjutkan ke tim onboarding supaya prosesnya cepat dan rapi."

Larangan:

* jangan muter di "lihat dulu step"
* jangan tutup dengan edukasi umum saat user sudah HOT
* jangan handoff tanpa menjelaskan kenapa
